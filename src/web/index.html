Dog or Cat?








This website demonstrates the use of a convolutional neural network to classify images of cats and dogs. The model was built with the high-level neural network package called Keras and trained with this Kaggle dataset, which contains tens of thousands of dog and cat images.

Click the "Invoke" button to run the classifier. Press "Random" to run it on a random Flickr Image.
